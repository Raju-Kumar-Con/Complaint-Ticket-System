# Complaint Ticket System

A Complaint Ticket Management System developed using ASP.NET Core MVC where users can register, log in, raise complaints, and administrators can manage, assign, and track complaint status.

---

## 🚀 Technologies Used

- ASP.NET Core MVC
- C#
- ADO.NET
- SQL Server
- Bootstrap 5
- jQuery
- AJAX
- HTML5
- CSS3

---

## ✨ Features

### User
- User Registration
- Secure Login & Logout
- Raise Complaint
- Edit Complaint
- View Complaint Details
- Complaint History
- Email Notification on Complaint Creation
- Profile Image Upload

### Admin
- Dashboard
- Complaint Management
- Assign Complaint to Support
- Update Complaint Status
- Manage Categories
- User Management
- Reports
- AJAX Search
- AG Grid Integration

### Support
- View Assigned Complaints
- Update Complaint Status

## ⚙ Setup

1. Clone the repository
2. Open the solution in Visual Studio
3. Restore NuGet packages
4. Execute the SQL script
5. Update the Connection String in **appsettings.json**
6. Configure SMTP settings
7. Run the project
